# Discord Wenz

A beautiful Discord-themed web project with galaxy background particles and modern styling.

## Features

- 🌌 Galaxy background with animated particles
- 💎 Modern dark theme with cyan accents
- 📱 Fully responsive design
- ✨ Smooth animations and transitions
- 🎨 Beautiful gradient backgrounds

## Installation

1. Clone this repository
```bash
git clone https://github.com/yourusername/discord-wenz.git
```

2. Navigate to the project folder
```bash
cd discord-wenz
```

3. Open `index.html` in your browser

## File Structure

```

├── index.html          # Main HTML file

```

- HTML5
- CSS3

## Customization

- Edit `index.html` to add your content
- Modify `style.css` to change colors and styling
- Customize particles count by adding/removing particle divs
